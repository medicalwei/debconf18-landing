DebConf 來台灣

第十九屆國際 Debian 開發者大會

2018 年的 DebConf 將在台灣的國立交通大學舉辦。

The Debian Project 是國際知名的大型自由軟體計畫，而 DebConf 是該計畫的年度開發 者大會。

Debian 是個非營利、不限定特定廠商的自由軟體系統。DebConf 讓全世界的志願開發者， 一年一度聚集，並集中在 Debian 發行版的開發與改進上。
